# Спецификация супер-агента «Печень как PBPK-агент»

Референсный пример PBPK-подхода в мультиагентной системе. Печень — **мост между физиологией и фармакокинетикой**, поэтому её спецификация отличается от ССС и ГИС: акцент на транспорте веществ, метаболизме и персонализации на уровне ферментов.

---

## 1. Роль и ответственность

**Супер-агент печени** — центральный узел метаболизма, детоксикации и синтеза в организме. Обрабатывает всё, что проходит через кровоток: нутриенты из ЖКТ, лекарства, эндогенные метаболиты, гормоны, ксенобиотики.

**Ключевые функции**:
1. **First-pass метаболизм** — обработка веществ из воротной вены до попадания в системный кровоток.
2. **Биотрансформация ксенобиотиков** — CYP450 (фаза I), конъюгация (фаза II).
3. **Билиарная экскреция** — выведение веществ и метаболитов в желчь.
4. **Энтерогепатическая циркуляция** — возврат веществ из ЖКТ через воротную вену.
5. **Гомеостаз глюкозы** — гликогенез, гликогенолиз, глюконеогенез (пересекается с ГИС).
6. **Липидный обмен** — синтез VLDL, холестерин, β-окисление жирных кислот.
7. **Белковый синтез** — альбумин, факторы свёртывания, транспортные белки (TBG, SHBG, трансферрин).
8. **Метаболизм гормонов** — T4 → T3 (дейодиназа), инактивация стероидов, инсулина.
9. **Депонирование** — гликоген (до 100 г), витамин A, B12, железо (ферритин).
10. **Детоксикация аммиака** — цикл мочевины.

**Границы ответственности**:
- ✅ Метаболизм веществ, транспорт через гепатоциты, синтез белков плазмы
- ✅ First-pass эффект, энтерогепатическая циркуляция
- ❌ Пищеварение (это к ЖКТ; печень получает уже всасывающиеся вещества)
- ❌ Иммунная функция в полной мере (Купферовские клетки — упрощённо)
- ❌ Гемодинамика (это к ССС; печень потребляет кровоток, но не управляет им)

---

## 2. Внутренняя иерархия (12 субагентов)

Печень имеет **уникальную двойную архитектуру**: функционально она разделена на **3 зоны Раппапорта** (по оксигенации), а структурно — на **клеточные компартменты** (по локализации ферментов).

```
Liver_SuperAgent (корень)
│
├── Vascular_Space_Agent              # кровь внутри печени (12% объёма)
│   ├── Sinusoidal_Blood              # смешанная кровь в синусоидах
│   └── Protein_Binding               # альбумин, α1-AGP, связывание лекарств
│
├── Hepatocyte_Zone1_Agent            # пери-портальная зона (высокий O₂)
│   ├── CYP3A4, CYP2C9, CYP2C19       # оксидативный метаболизм
│   ├── Gluconeogenesis               # глюкогенез при голодании
│   ├── Beta_Oxidation                # жирные кислоты
│   └── Urea_Cycle                    # цикл мочевины (преимущественно здесь)
│
├── Hepatocyte_Zone2_Agent            # переходная зона
│   ├── Смешанный фенотип
│   └── Промежуточный метаболизм
│
├── Hepatocyte_Zone3_Agent            # пери-центральная зона (низкий O₂)
│   ├── CYP2E1                        # этанол, парацетамол → NAPQI
│   ├── CYP1A2                        # кофеин, теофиллин
│   ├── Glycolysis                    # гликолиз
│   ├── Lipogenesis                   # синтез жирных кислот
│   └── Glutathione_System            # детоксикация NAPQI
│
├── Biliary_Space_Agent               # желчные каналы (1.5% объёма)
│   ├── Bile_Acid_Pool                # синтез из холестерина
│   ├── Biliary_Excretion             # транспортёры MRP2, BCRP, BSEP
│   └── Enterohepatic_Circulation     # возврат через ЖКТ
│
├── Transport_System_Agent            # мембранные транспортёры
│   ├── Uptake: OATP1B1, OATP1B3, NTCP, OCT1
│   └── Efflux: P-gp (MDR1), BCRP, MRP2, MRP3
│
├── Kupffer_Cell_Agent                # резидентные макрофаги (упрощённо)
│   └── Cytokine_Secretion (TNF-α, IL-6) → подавление CYP при воспалении
│
├── Stellate_Cell_Agent               # Ito-клетки (фиброз, витамин A)
│   └── Fibrogenesis (при хроническом повреждении)
│
├── Storage_Agent                     # депо
│   ├── Glycogen_Store (до 100 г)
│   ├── Vitamin_A_Store
│   ├── Iron_Ferritin
│   └── Vitamin_B12
│
├── Protein_Synthesis_Agent           # синтез белков плазмы
│   ├── Albumin
│   ├── Coagulation_Factors (II, VII, IX, X, fibrinogen)
│   ├── Transport_Proteins (TBG, SHBG, transferrin, ceruloplasmin)
│   └── CRP (острая фаза)
│
├── Hormone_Metabolism_Agent          # инактивация гормонов
│   ├── 5'-Deiodinase (T4 → T3)
│   ├── Steroid_Inactivation (кортизол → кортизон)
│   └── Insulin_Clearance (~50% первого прохождения)
│
└── Blood_Flow_Interface              # стык с ССС
    ├── Hepatic_Artery_Input          # 25% кровотока, O₂-богатая
    ├── Portal_Vein_Input             # 75% кровотока, из ЖКТ
    └── Hepatic_Vein_Output           # в нижнюю полую вену
```

**Итого**: 1 супер-агент + 12 функциональных субагентов. Это **средний уровень детализации** — оптимум для цифрового двойника.

### Уровни детализации (на выбор)

| Уровень | Субагентов | Когда использовать |
|---|---|---|
| **L1: Well-stirred** | 1 | MVP, быстрый прототип |
| **L2: Zonal (3 зоны)** | 12 (как выше) | **Рекомендуется** для цифрового двойника |
| **L3: Cellular** | 30–50 | Токсикология, редкие болезни, исследования |

## 3. Интерфейс наружу (контракт супер-агента)

### 3.1. Входы

| Источник | Сигнал | Частота | Назначение |
|---|---|---|---|
| CVS (артерия) | `C_arterial[substance]` | 0.1 Hz | Вещества из системного кровотока |
| Gut (воротная вена) | `C_portal[substance]`, `bile_acids` | 0.1 Hz | Всасывающиеся вещества, нутриенты |
| ГИС | `insulin`, `glucagon`, `glucose` | 0.1 Hz | Регуляция гликогена, глюконеогенеза |
| Эндокринная | `cortisol`, `T3/T4`, `growth_hormone`, `thyroxine` | 0.01 Hz | Метаболическая регуляция |
| Иммунная | `cytokines (IL-6, TNF-α)` | 0.001 Hz | Подавление CYP при воспалении |
| Orchestrator | `drug_administration{substance, dose, route}` | event | Фармакокинетика |
| Orchestrator | `ethanol_intake_g` | event | Метаболизм этанола |
| Nervous | `sympathetic_tone` | 0.1 Hz | Влияние на кровоток печени |
| Внутренний | `patient_genotype{CYP2D6, CYP2C19, ...}` | init | Персонализация ферментов |

### 3.2. Выходы

| Сигнал | Подписчики | Частота |
|---|---|---|
| `C_hepatic_vein[substance]` | CVS | 0.1 Hz |
| `metabolites[substance]` | CVS (для дальнейшей экскреции) | 0.1 Hz |
| `bile_excretion[substance]` | Gut (в ЖКТ) | 0.1 Hz |
| `glucose_output_g_min` | ГИС (Blood_Pool) | 0.1 Hz |
| `albumin_g_L`, `coagulation_factors` | CVS | 0.001 Hz |
| `T3_production_pmol_min` | Эндокринная | 0.001 Hz |
| `urea_production_mmol_min` | Почки | 0.01 Hz |
| `liver_function_score` (Child-Pugh прокси) | UI, Orchestrator | 0.001 Hz |
| `liver_status_summary` | UI | 0.01 Hz |

### 3.3. Интерфейс агента

```python
class LiverPBPKSuperAgent(Agent):
    state: LiverState  # ~150 полей
    
    # Реестр веществ, которые печень умеет обрабатывать
    substances: Dict[SubstanceID, SubstancePKProfile]
    
    def step(self, t, dt, inbox) -> StepResult:
        # 1. Получить inflows из артерии и воротной вены
        # 2. Для каждого вещества в крови:
        #    a. Uptake через транспортёры (с учётом генотипа)
        #    b. Внутриклеточный метаболизм по зонам
        #    c. Efflux обратно в кровь или в желчь
        # 3. Обновить глюкозу/гликоген/липиды
        # 4. Синтез белков (медленный контур)
        # 5. Обновить функцию Купферовских клеток (цитокины)
        # 6. Сформировать outflows
    
    def calibrate(self, patient) -> None:
        # Генотипы CYP, Child-Pugh, NAFLD, масса печени
    
    def validate(self, benchmark) -> ValidationReport:
        # PK-бенчмарки для 10+ веществ
```

## 5. Порядок обновления субагентов

```
Внутри step(t, dt):

  1. Blood_Flow_Interface.update()
     // Получить Q_HA, Q_PV, концентрации на входе
     // Смешать: C_in = 0.25·C_arterial + 0.75·C_portal

  2. Vascular_Space_Agent.update(C_in, protein_binding)
     // Обновить связывание с белками плазмы
     // → f_u (свободная фракция) для каждого вещества

  3. Transport_System_Agent.update(f_u, transporters_activity)
     // Рассчитать uptake в гепатоциты:
     // J_uptake = CL_uptake · f_u · C_plasma
     // С учётом генотипа (OATP1B1 polymorphism)

  4. Hepatocyte_Zone1_Agent.update(C_intracellular, O2_high)
     // CYP3A4, CYP2C9, CYP2C19 метаболизм
     // Глюконеогенез (если инсулин низкий)
     // Цикл мочевины

  5. Hepatocyte_Zone2_Agent.update(C_intracellular, O2_medium)
     // Промежуточный метаболизм

  6. Hepatocyte_Zone3_Agent.update(C_intracellular, O2_low)
     // CYP2E1 (этанол, парацетамол)
     // CYP1A2 (кофеин)
     // Липогенез
     // Глутатион-система (детоксикация NAPQI)

  7. Biliary_Space_Agent.update(metabolites, bile_acids)
     // Экскреция через MRP2, BCRP, BSEP
     // Формирование желчи

  8. Kupffer_Cell_Agent.update(cytokines)
     // Если IL-6/TNF-α ↑ → downregulation CYP (коэффициент 0.5–0.8)

  9. Storage_Agent.update(glycogen, vitamins, iron)
     // Баланс депо

  10. Protein_Synthesis_Agent.update_if_due(t)
      // Раз в час: синтез альбумина, факторов свёртывания
      // (медленный контур, T½ альбумина = 20 дней)

  11. Hormone_Metabolism_Agent.update()
      // T4 → T3, инсулин → инактивация, кортизол → кортизон

  12. Self.state = aggregate(all subagents)
  13. Publish outflows (hepatic vein, bile, glucose)
```

## 7. Математическое ядро

### 7.1. Баланс вещества в печени (общий)

```
dA_liver/dt = Q_HA·C_art + Q_PV·C_port - Q_HV·C_hv
              - CL_uptake·f_u·C_plasma
              + CL_efflux_blood·C_hep
              - CL_bile·C_hep
              - CL_met·C_hep
```

### 7.2. Well-stirred модель (упрощение для быстрой симуляции)

```
E_H = CL_int · f_u / (Q_H + CL_int · f_u)    # extraction ratio
F_H = 1 - E_H                                  # биодоступность через печень
CL_H = Q_H · E_H                               # печёночный клиренс
```

### 7.3. Метаболизм CYP450 (Michaelis-Menten)

```
v = V_max · C / (K_m + C)
```

Где `V_max = k_cat · [CYP] · [hepatocyte]`. Для ингибирования:

```
v = V_max · C / (K_m · (1 + I/K_i) + C)
```

### 7.4. Uptake через транспортёры

```
J_uptake = (V_max_uptake · C / (K_m_uptake + C)) + (PS_passive · C)
```

Для OATP1B1 (статины):
- Нормальный: V_max = 100%
- *5 аллель (Val174Ala): V_max = 40% → ↑ концентрация статина в крови → ↑ риск миопатии

### 7.5. Зональная гетерогенность

```
C_zone1 = C_in
C_zone2 = C_zone1 · exp(-CL_zone1 / Q_per_zone)
C_zone3 = C_zone2 · exp(-CL_zone2 / Q_per_zone)
C_hv    = C_zone3 · exp(-CL_zone3 / Q_per_zone)
```

Это даёт **градиент концентрации** от портальной зоны к центральной — ключевой для токсичности (зона 3 страдает первой при гипоксии/токсинах).

```
│   ├── Uptake: OATP1B1, OATP1B3, NTCP, OCT1
│   └── Efflux: P-gp (MDR1), BCRP, MRP2, MRP3

│   └── Cytokine_Secretion (TNF-α, IL-6) → подавление CYP при воспалении
│
├── Stellate_Cell_Agent               # Ito-клетки (фиброз, витамин A)
│   └── Fibrogenesis (при хроническом повреждении)

│   ├── Glycogen_Store (до 100 г)
│   ├── Vitamin_A_Store
│   ├── Iron_Ferritin
│   └── Vitamin_B12
│
├── Protein_Synthesis_Agent           # синтез белков плазмы
│   ├── Albumin
│   ├── Coagulation_Factors (II, VII, IX, X, fibrinogen)
│   ├── Transport_Proteins (TBG, SHBG, transferrin, ceruloplasmin)
│   └── CRP (острая фаза)
│
├── Hormone_Metabolism_Agent          # инактивация гормонов
│   ├── 5'-Deiodinase (T4 → T3)
│   ├── Steroid_Inactivation (кортизол → кортизон)
│   └── Insulin_Clearance (~50% первого прохождения)
│
└── Blood_Flow_Interface              # стык с ССС
    ├── Hepatic_Artery_Input          # 25% кровотока, O₂-богатая
    ├── Portal_Vein_Input             # 75% кровотока, из ЖКТ
    └── Hepatic_Vein_Output           # в нижнюю полую вену
```

**Итого**: 1 супер-агент + 12 функциональных субагентов. Это **средний уровень детализации** — оптимум для цифрового двойника.

## 5. Порядок обновления субагентов

```
Внутри step(t, dt):

  1. Blood_Flow_Interface.update()
     // Получить Q_HA, Q_PV, концентрации на входе
     // Смешать: C_in = 0.25·C_arterial + 0.75·C_portal

  2. Vascular_Space_Agent.update(C_in, protein_binding)
     // Обновить связывание с белками плазмы
     // → f_u (свободная фракция) для каждого вещества

  3. Transport_System_Agent.update(f_u, transporters_activity)
     // Рассчитать uptake в гепатоциты:
     // J_uptake = CL_uptake · f_u · C_plasma
     // С учётом генотипа (OATP1B1 polymorphism)

  4. Hepatocyte_Zone1_Agent.update(C_intracellular, O2_high)
     // CYP3A4, CYP2C9, CYP2C19 метаболизм
     // Глюконеогенез (если инсулин низкий)
     // Цикл мочевины

  5. Hepatocyte_Zone2_Agent.update(C_intracellular, O2_medium)
     // Промежуточный метаболизм

  6. Hepatocyte_Zone3_Agent.update(C_intracellular, O2_low)
     // CYP2E1 (этанол, парацетамол)
     // CYP1A2 (кофеин)
     // Липогенез
     // Глутатион-система (детоксикация NAPQI)

  7. Biliary_Space_Agent.update(metabolites, bile_acids)
     // Экскреция через MRP2, BCRP, BSEP
     // Формирование желчи

  8. Kupffer_Cell_Agent.update(cytokines)
     // Если IL-6/TNF-α ↑ → downregulation CYP (коэффициент 0.5–0.8)

  9. Storage_Agent.update(glycogen, vitamins, iron)
     // Баланс депо

  10. Protein_Synthesis_Agent.update_if_due(t)
      // Раз в час: синтез альбумина, факторов свёртывания
      // (медленный контур, T½ альбумина = 20 дней)

  11. Hormone_Metabolism_Agent.update()
      // T4 → T3, инсулин → инактивация, кортизол → кортизон

  12. Self.state = aggregate(all subagents)
  13. Publish outflows (hepatic vein, bile, glucose)
```

## 7. Математическое ядро

### 7.1. Баланс вещества в печени (общий)

```
dA_liver/dt = Q_HA·C_art + Q_PV·C_port - Q_HV·C_hv
              - CL_uptake·f_u·C_plasma
              + CL_efflux_blood·C_hep
              - CL_bile·C_hep
              - CL_met·C_hep
```

### 7.2. Well-stirred модель (упрощение для быстрой симуляции)

```
E_H = CL_int · f_u / (Q_H + CL_int · f_u)    # extraction ratio
F_H = 1 - E_H                                  # биодоступность через печень
CL_H = Q_H · E_H                               # печёночный клиренс
```

### 7.3. Метаболизм CYP450 (Michaelis-Menten)

```
v = V_max · C / (K_m + C)
```

Где `V_max = k_cat · [CYP] · [hepatocyte]`. Для ингибирования:

```
v = V_max · C / (K_m · (1 + I/K_i) + C)
```

### 7.4. Uptake через транспортёры

```
J_uptake = (V_max_uptake · C / (K_m_uptake + C)) + (PS_passive · C)
```

Для OATP1B1 (статины):
- Нормальный: V_max = 100%
- *5 аллель (Val174Ala): V_max = 40% → ↑ концентрация статина в крови → ↑ риск миопатии

### 7.5. Зональная гетерогенность

```
C_zone1 = C_in
C_zone2 = C_zone1 · exp(-CL_zone1 / Q_per_zone)
C_zone3 = C_zone2 · exp(-CL_zone2 / Q_per_zone)
C_hv    = C_zone3 · exp(-CL_zone3 / Q_per_zone)
```

Это даёт **градиент концентрации** от портальной зоны к центральной — ключевой для токсичности (зона 3 страдает первой при гипоксии/токсинах).

### 7.6. Подавление CYP цитокинами

```
CYP_activity = CYP_baseline · (1 - I_max · IL6^n / (IC50^n + IL6^n))
```

При остром воспалении — CYP ↓ на 30–70%, что меняет фармакокинетику лекарств.

### 7.7. Глюконеогенез (пересечение с ГИС)

```
GNG = GNG_max · (1 + k_gluca · Glucagon) · (1 - k_ins · Insulin) · (1 + k_cort · Cortisol)
```

## 8. Примеры сценариев

### 8.1. First-pass эффект: пропранолол 40 мг перорально

```
t=0      Пропранолол принят, всасывается в ЖКТ
         C_portal ↑ до 5 µM (высокая, т.к. воротная вена собирает всё)

t=5 мин  Печень (зона 1): CYP2D6 + CYP1A2 метаболизирует
         Extraction ratio E_H = 0.70 (высокий)
         F_H = 0.30 (только 30% проходит в системный кровоток)

t=30 мин C_arterial (системная) = 0.3 µM (пик)
         C_max системный = 0.3 µM
         AUC_systemic = 0.3 × 8 = 2.4 µM·h

         Если бы тот же болюс в/в:
         C_max = 1.0 µM, AUC = 8.0 µM·h
         → Биодоступность F = 2.4/8.0 = 0.30 ✓

Результат: модель правильно воспроизводит first-pass эффект.
```

### 8.2. DDI: грейпфрутовый сок + симвастатин

```
t=0      Пациент принимает симвастатин 20 мг
         + 300 мл грейпфрутового сока (ингибитор CYP3A4 и OATP)

t=30 мин Грейпфрутовый сок: фуранокумарины необратимо ингибируют
         кишечный CYP3A4 и OATP1B1 в печени

t=2 ч    Симвастатин:
         - Без грейпфрута: C_max = 5 нг/мл, AUC = 15 нг·ч/мл
         - С грейпфрутом: C_max = 25 нг/мл (5×), AUC = 75 нг·ч/мл
         
         → Риск миопатии и рабдомиолиза ↑

Модель должна показать:
  - ↓ CL_int CYP3A4 на 70% (кишечник) + на 30% (печень)
  - ↓ OATP1B1 uptake на 50%
  - Итоговое ↑ AUC в 4–5×
```

### 8.3. Парацетамол: дозозависимая токсичность

```
Сценарий A: 1 г (терапевтическая доза)
  - 90% → глюкуронидация + сульфатирование (безопасно)
  - 10% → CYP2E1 → NAPQI
  - NAPQI детоксифицируется глутатионом (GSH)
  - Результат: безопасно

Сценарий B: 10 г (передозировка)
  - Пути конъюгации насыщены (Km достигнут)
  - 30% → CYP2E1 → NAPQI (избыток)
  - GSH истощается (< 20% от нормы)
  - NAPQI связывается с белками гепатоцитов → некроз зоны 3
  - ALT/AST ↑ в 100× через 48–72 ч
  - Риск острой печёночной недостаточности

Модель должна предсказать:
  - Порог токсичности: ~7–10 г для взрослого
  - Временную динамику ALT/AST
  - Влияние алкоголя (индукция CYP2E1 → ↓ порог)
  - Влияние голодания (↓ GSH → ↓ порог)
```

### 8.4. Цирроз печени (Child-Pugh B)

```
Параметры пациента:
  - Альбумин: 28 г/л (↓)
  - Билирубин: 45 µM (↑)
  - МНО: 1.6 (↑)
  - Асцит: умеренный
  - Энцефалопатия: 1–2 ст.

Изменения в модели:
  - Q_PV ↓ на 40% (портальная гипертензия)
  - Функциональная масса гепатоцитов ↓ на 50%
  - Альбумин-синтез ↓ на 60%
  - CYP-активность ↓ на 50% (общий клиренс ↓)
  - First-pass эффект ↓ → F_H ↑ для лекарств с высокой экстракцией
  
Результат для пропранолола:
  - F_H увеличивается с 0.30 до 0.60
  - C_max в 2× выше
  - Дозу нужно ↓ на 50%
```

### 11.2. Проекция на параметры

```python
def calibrate(passport) -> LiverParameters:
    # 1. Масса печени (по весу и полу)
    liver_mass = 0.021 * passport.weight_kg  # ~2.1% от массы тела
    if passport.sex == 'F':
        liver_mass *= 0.90
    
    # 2. Функциональный статус (Child-Pugh прокси)
    if passport.liver_disease.startswith('cirrhosis'):
        functional_mass *= {
            'cirrhosis_A': 0.70,
            'cirrhosis_B': 0.50,
            'cirrhosis_C': 0.30
        }[passport.liver_disease]
        Q_PV *= {
            'cirrhosis_A': 0.80,
            'cirrhosis_B': 0.60,
            'cirrhosis_C': 0.40
        }[passport.liver_disease]
    
    # 3. Генотипы CYP → активность ферментов
    cyp2d6_activity = {
        '*1/*1': 1.0,     # extensive metabolizer
        '*1/*4': 0.5,     # intermediate
        '*4/*4': 0.05,    # poor metabolizer
        '*1/*1xN': 2.0    # ultra-rapid
    }[passport.CYP2D6_genotype]
    
    cyp2c19_activity = {
        '*1/*1': 1.0,
        '*1/*2': 0.5,
        '*2/*2': 0.05,
        '*17/*17': 1.5
    }[passport.CYP2C19_genotype]
    
    oatp1b1_activity = {
        '*1a/*1a': 1.0,
        '*1a/*5': 0.5,
        '*5/*5': 0.2
    }[passport.OATP1B1_genotype]
    
    # 4. NAFLD/NASH → стеатоз, воспаление
    if passport.liver_disease in ['NAFLD', 'NASH']:
        lipid_accumulation = 0.15 if NAFLD else 0.25
        cyp_activity *= 0.85  # лёгкое снижение
        inflammation_IL6 = 2.0 if NASH else 0.5
    
    # 5. Алкоголь → индукция CYP2E1
    cyp2e1_activity = 1.0 + 0.5 * min(passport.alcohol_g_day / 40, 2.0)
    
    # 6. Воспаление → подавление CYP
    if passport.inflammation_CRP_mg_L > 10:
        cytokine_suppression = 0.7  # CYP ↓ на 30%
    
    # 7. Возраст → снижение кровотока и массы
    age_factor = 1.0 - 0.005 * max(0, passport.age - 50)
    
    return LiverParameters(
        liver_mass=liver_mass,
        functional_mass=functional_mass,
        Q_HA=0.36 * age_factor,
        Q_PV=1.08 * age_factor,
        cyp3a4_activity=1.0,
        cyp2d6_activity=cyp2d6_activity,
        cyp2c19_activity=cyp2c19_activity,
        cyp2e1_activity=cyp2e1_activity,
        oatp1b1_activity=oatp1b1_activity,
        albumin_synthesis=10.0 * (passport.albumin_g_L / 40),
        # ...
    )
```

## 12. Интеграция с другими системами

| Система | Механизм интеграции |
|---|---|
| **ССС** | Печень потребляет 25% CO. Артерия/воротная вена/печёночная вена — общие с CVS. |
| **ГИС** | Глюкоза, гликоген, глюконеогенез — общие параметры. Liver_Agent ГИС может стать частью LiverPBPK как подмодуль. |
| **ЖКТ** | Воротная вена — вход из ЖКТ. Желчь — выход в ЖКТ. Энтерогепатическая циркуляция. |
| **Эндокринная** | Метаболизм T4→T3, стероидов, инсулина. Синтез TBG, SHBG. |
| **Почки** | Мочевина из печени → почки. Метаболиты в крови → почечная экскреция. |
| **Иммунная** | Цитокины IL-6, TNF-α → подавление CYP. Купферовские клетки. |
| **Жировая ткань** | VLDL-секреция, липопротеины, NAFLD. |

**Ключевая архитектурная идея**: печень — это **хаб метаболизма**. Все вещества, которые проходят через организм, рано или поздно оказываются в печени. Поэтому PBPK-слой печени должен быть **реестром всех веществ** с их параметрами (logP, pKa, CYP-субстраты, транспортёры).
