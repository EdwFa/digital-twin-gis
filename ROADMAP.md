# 🗺 Дорожная карта (Roadmap) проекта "Цифровой двойник"

Данный документ описывает долгосрочный план развития 3-уровневой мультиагентной системы.

## ✅ Фаза 1: Макро-MVP (Завершено)
*Цель: Доказать жизнеспособность математической модели и EventBus на примере одной системы.*
- [x] Создание ядра (Simulation Engine, Blood Pool, Message Bus).
- [x] Базовые макро-уравнения для поджелудочной, печени, мышц, кишечника, жировой ткани, почек и мозга.
- [x] Паспорт пациента и начальная калибровка (HOMA-IR, BMI).
- [x] Интерактивный UI дашборд (Streamlit).
- [x] Базовый Validation Suite (OGTT для здорового и T2D пациента).

## ⏳ Фаза 2: Глубокая микро-симуляция (В процессе)
*Цель: Углубить Глюкозо-инсулиновую систему (GIS) до микро-процессов (Уровень 2).*
- [x] 1. Добавление `Incretin_Pool` и `FFA` (свободных жирных кислот) в ядро крови.
- [ ] 2. **Gut_Agent**: Моделирование L/K клеток и выработки инкретинов (GLP-1) пропорционально всасыванию углеводов.
- [ ] 3. **Pancreas_Agent**: Разделение на Alpha, Beta (2-pool модель: RRP и Reserve) и Delta клетки (соматостатин, паракринное подавление).
- [ ] 4. **Liver_Agent**: Трекинг пула гликогена (~100г), разделение на глюконеогенез и гликогенолиз.
- [ ] 5. **Muscle_Agent**: Трекинг пула гликогена (~400г), эффект насыщения GLUT4.
- [ ] 6. **Adipose_Agent**: Моделирование липолиза и секреции адипокинов (лептин, адипонектин).
- [ ] 7. **Slow_Adaptation_Agent**: Динамическая гибель бета-клеток в реальном времени от глюкотоксичности.

## 📅 Фаза 3: Масштабирование систем (Уровень 1)
*Цель: Добавить оставшиеся 10 биологических систем и настроить кросс-системную интеграцию.*
- [ ] Сердечно-сосудистая система (гемодинамика, барорецепторы).
- [ ] Нервная система (ЦНС, парасимпатика, симпатика).
- [ ] Эндокринная система (кортизол, щитовидная железа).
- [ ] Дыхательная система (сатурация O2, CO2).
- [ ] Полноценная выделительная система.
- [ ] Иммунная система и симуляция воспаления.
- [ ] Опорно-двигательная, терморегуляция, пищеварительная, репродуктивная.
- [ ] Кросс-системные тесты (например, влияние кортизола (стресса) на пульс и выброс глюкозы).

## 🚀 Фаза 4: Стратегический оркестратор (Уровень 0)
*Цель: Интеграция "Мозга/Врача", планировщика и AI-ассистента.*
- [ ] Интерфейс Life Timeline (расписание сна, еды, тренировок на месяцы вперед).
- [ ] Виртуальная фармакология (назначение Метформина, Инсулина, Ингибиторов SGLT-2).
- [ ] LLM-интеграция: Анализ кривых искусственным интеллектом и выдача текстовых клинических рекомендаций.
- [ ] CI/CD для постоянного прогона медицинских гипотез (Continuous Clinical Validation).
